# LM Calc

LM Calc is a fullstack Angular + .NET 9 application deployed on Azure.

## Architecture
- **UI (Angular)** → Azure Static Web Apps
- **Backend (.NET 9)** → AKS (containerized with Docker)
- **API Gateway** → Azure API Management
- **Environments**: QA & Prod
